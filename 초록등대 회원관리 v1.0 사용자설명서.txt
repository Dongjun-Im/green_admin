초록등대 회원관리 v1.0 사용자 설명서 (안정 버전)
==========================================

안녕하세요.
이 설명서는 스크린리더로 한 줄씩 편하게 읽으실 수 있게
짧은 문장과 쉬운 말로 썼어요.


==========================================
1. 이 프로그램은 무엇인가요
==========================================

초록등대 동호회의 회원 관리를 도와주는 프로그램이에요.
사람이 일일이 손으로 하던 일을 버튼 한 번으로 끝낼 수 있어요.

이 프로그램이 해 주는 일:

  1. 우수회원 명단을 파일로 저장 (3개월마다)
  2. 오래 안 들어온 회원 등급 정리 (6개월마다)
  3. 글을 많이 쓴 회원을 상위 등급으로 올려줌
  4. 회원을 이름이나 아이디로 찾아보기
  5. 회원들에게 안내 메일 보내기


==========================================
2. 누가 쓸 수 있나요
==========================================

초록등대 동호회의 "동호회 관리자"만 쓸 수 있어요.

일반 회원이나 우수 회원 아이디로 들어가면
프로그램이 열리지 않고 바로 닫혀요. 안심하세요.


==========================================
3. 프로그램 실행하기
==========================================

방법 1 - 실행 파일을 더블클릭

  파일 탐색기에서 이 경로로 가세요.
  Y:\Downloads\자작 프로그램\초록등대회원관리\dist\초록등대회원관리

  이 안에 "초록등대회원관리.exe" 라는 파일이 있어요.
  더블클릭하면 프로그램이 시작돼요.

방법 2 - cmd창에서 실행

  cmd창에 아래 세 줄을 차례로 입력하세요.

  Y:
  cd "\Downloads\자작 프로그램\초록등대회원관리\dist\초록등대회원관리"
  초록등대회원관리.exe


==========================================
4. 처음 실행했을 때
==========================================

프로그램을 켜면 로그인 창이 떠요.

  1. 소리샘 아이디를 입력하세요.
  2. Tab 키를 누르고 비밀번호를 입력하세요.
  3. Tab 키를 누르면 "아이디와 비밀번호 저장" 체크박스가 있어요.
     Space 키로 체크하세요. 다음부터는 자동으로 로그인돼요.
  4. Tab 키를 누르고 엔터 키로 확인 버튼을 누르세요.

로그인이 성공하면 "인증에 성공했습니다" 라는 음성이 나와요.
잠시 기다리면 메인 창이 열려요.


==========================================
5. 메인 창에서 할 수 있는 일
==========================================

메인 창이 뜨면 화면 가운데에 두 개의 글상자가 있어요.

  "현재 상태" 상자
      마지막으로 백업한 날짜, 다음 백업일,
      마지막으로 조정한 날짜, 다음 조정일을 보여줘요.

  "작업 결과" 상자
      방금 끝낸 작업의 결과가 여기에 남아요.
      실수로 놓쳐도 다시 여기서 읽을 수 있어요.

메뉴바에는 파일, 작업, 개발자, 도움말 메뉴가 있어요.
가장 많이 쓰는 건 "작업" 메뉴예요.


==========================================
6. 꼭 기억할 단축키
==========================================

모든 기능은 단축키로 쓸 수 있어요.
외우기 쉽게 만들었어요.

  Ctrl + F    회원 찾기 (Find)
  Ctrl + G    (검색 창에서) 회원 등급 변경 (Grade)
  Ctrl + N    (검색 창에서) 회원 메모/태그 편집 (Note)
  Ctrl + T    회원 통계 (sTats) — 등급별 인원과 최근 활동
  Ctrl + Z    마지막 작업 되돌리기 (Undo)
  Ctrl + Shift + Y  등급 변경 이력 (hYstory) — 영구 보관 (v1.0)
  Ctrl + Shift + D  백업 비교 (Diff) — 분기 간 변동 한눈에
  Ctrl + Shift + L  작업 로그 뷰어 (Log)
  Ctrl + B    우수회원 백업 (Backup)
  Ctrl + U    자동 승급 (Upgrade)
  Ctrl + R    장기 미접속 조정 - 미리보기 (Review)
  Ctrl + M    수동 메일 보내기 (Mail, rtgreen 전용)
  Ctrl + I    마지막 작업일 음성 안내 (Info)
  Ctrl + D    지금 작업할 수 있는지 확인 (Due)
  Ctrl + O    백업 폴더 열기 (Open)
  Ctrl + L    로그아웃 (Logout)
  Ctrl + K    단축키 안내 창
  F1          프로그램 정보
  Alt + F4    프로그램 종료
  Esc         음성 멈추기


==========================================
7. 기능 하나씩 쉽게 알려드려요
==========================================

----------------------------------------
회원 찾기 (Ctrl + F)
----------------------------------------

어떤 회원이 있는지, 그 회원의 등급이 뭔지 알고 싶을 때 써요.

사용 방법:

  1. Ctrl + F 를 누르면 검색 창이 열려요.
  2. 검색창에 이름, 아이디, 닉네임 중 일부만 적어요.
     예) "홍길동" 이라고 적으면 홍길동, 홍길순 같은
         이름 안에 "홍길동" 이 들어간 모든 회원이 나와요.
  3. 아래 화살표로 회원 목록을 훑어 볼 수 있어요.
  4. 회원을 선택하면 그 아래에 상세 정보가 나와요.
     아이디, 이름, 닉네임, 등급, 마지막 접속일, 가입일.
  5. 다 봤으면 Esc 키로 닫으세요.

처음 검색할 때는 전체 회원 목록을 받아오느라
잠깐 기다려야 해요. 그 다음부터는 바로 바로 열려요.


----------------------------------------
회원 등급 직접 변경 (v0.4 신규 — 검색 창에서 Ctrl + G 또는 더블클릭)
----------------------------------------

특정 회원 한 명의 등급을 관리자가 직접 올리거나 내리는 기능이에요.
자동 승급 규칙으로 처리되지 않는 예외 상황(예: 신청 회원을 바로
일반회원으로 올리기, 명예회원을 다시 우수회원으로 내리기 등)에 써요.

사용 순서:

  1. Ctrl + F 로 회원 검색 창을 엽니다.
  2. 검색해서 대상 회원을 위/아래 화살표로 선택해요.
  3. Ctrl + G 를 누르거나 회원을 더블클릭하면
     "등급 변경" 창이 열려요.
  4. 새 등급을 콤보박스에서 선택하세요.
       - 레벨 5 : 준회원
       - 레벨 6 : 일반회원
       - 레벨 7 : 우수회원
       - 레벨 8 : 최우수회원
       - 레벨 9 : 명예회원
  5. "변경" 버튼을 누르면 한 번 더 확인 창이 떠요.
     "예" 를 누르면 사이트에 즉시 반영돼요.

안전 장치:
  · 본인 계정의 등급은 이 화면에서 변경할 수 없어요.
    (실수로 관리자 권한을 잃지 않게 막는 안전 장치예요.)
  · 같은 등급으로 변경하려고 하면 "변경 필요 없음" 으로 처리돼요.
  · 모든 단건 변경은 로그 폴더에 "수동 승급/강등" 으로 기록돼요.


----------------------------------------
우수회원 백업 (Ctrl + B)
----------------------------------------

우수회원과 최우수회원의 명단을 파일로 저장해 주는 기능이에요.
3개월에 한 번씩 기록을 남기고 싶을 때 써요.

무슨 파일이 만들어지나요?

  텍스트 파일 (.txt)
      메모장이나 스크린리더로 읽기 좋게 만든 파일.
      줄마다 한 명씩 정리돼요.

  엑셀 파일 (.xlsx)
      엑셀이나 한글 오피스에서 열 수 있어요.
      나중에 정렬하거나 필터를 걸 수 있어요.

어디에 저장되나요?

  backups 폴더 안에 오늘 날짜로 새 폴더가 만들어지고
  그 안에 두 파일이 저장돼요.
  Ctrl + O 를 누르면 바로 백업 폴더가 열려요.

중요:
  이 버튼을 누르면 우수회원 백업뿐 아니라
  아래에 나오는 "자동 승급" 까지 함께 실행돼요.
  즉 한 번 누르면 두 가지 일이 자동으로 끝나요.


----------------------------------------
자동 승급 (Ctrl + U)
----------------------------------------

열심히 글을 쓴 회원에게 상위 등급을 자동으로 주는 기능이에요.
"우리들의 이야기" 게시판에 쓴 글 수를 세어서 판단해요.

승급 규칙 (v0.5 — 활동점수 기반):

활동점수가 무엇인가요?
  글 한 건 = 1.0 점, 댓글 한 건 = 0.3 점
  두 게시판(우리들의 이야기, 질문게시판)에서 쓴 모든 글과 댓글을 더해요.
  예) 글 5건 + 댓글 10건 = 5 + 3 = 8.0 점

승급 표:

  가입 대기 또는 신청 중인 회원이
  활동점수 3 이상이면
  → 준회원으로 승급돼요.

  준회원이
  활동점수 5 이상이면 → 일반회원으로 승급 (v0.5 신규)

  일반회원이
  활동점수 30 이상이면  → 우수회원으로 승급
  활동점수 60 이상이면  → 최우수회원으로 승급
  활동점수 120 이상이면 → 명예회원으로 승급

  ※ 댓글 검색을 사이트가 지원하지 않으면 댓글 수가 0으로 잡혀
     활동점수 = 글 수 와 같아져요. 그래도 정상 동작합니다.

중요한 점:

  이미 우수회원 이상인 분은 이 기능으로 낮아지지 않아요.
  올라가기만 하는 기능이에요.

  동호회 관리자와 본인은 언제나 제외돼요.

시간이 얼마나 걸리나요?

  전체 회원 중 일반회원과 가입 대기자를 한 명씩
  조회하기 때문에 1분에서 2분 정도 걸려요.
  조금만 기다려 주세요.


----------------------------------------
장기 미접속 조정 (Ctrl + R 그리고 Ctrl + Shift + R)
----------------------------------------

6개월 넘게 안 들어온 회원의 등급을 자동으로 조정하는 기능이에요.

조정 규칙 (v0.4 새 회원 체계):

  준회원     → 탈퇴
  일반회원   → 준회원
  우수회원   → 일반회원
  최우수회원 → 우수회원
  명예회원   → 조정 안 함 (그대로 유지)
  동호회관리자 권한자 → 조정 안 함 (그대로 유지)

사용 순서 (중요합니다):

  첫 번째 단계 - 미리보기 (Ctrl + R)

    회원 전체를 훑어서 "이 분들이 대상이에요" 하고
    목록만 보여주는 단계예요.
    사이트에는 아무것도 반영되지 않아요. 안전해요.

    미리보기 창이 열리면 위 아래 화살표로 한 명씩
    들어보세요. 음성으로 "아이디, 닉네임, 조정 내용" 이
    읽혀요.

    꼭 확인하세요:
      - 내가 아는 활동 회원이 잘못 들어있지 않나?
      - 본인이 목록에 있지는 않나? (본인은 자동으로 빠져요)

    TXT 내보내기 버튼을 누르면 목록을 메모장으로
    천천히 확인할 수도 있어요.

  두 번째 단계 - 실제 적용 (Ctrl + Shift + R 또는 적용 버튼)

    미리보기에서 확인을 마쳤다면 적용 버튼을 누르세요.
    "정말로 처리하시겠습니까?" 라는 확인 창이 한 번 더 떠요.
    여기서 "예" 를 누르면 실제로 사이트에 반영돼요.

    한 번 적용하면 되돌릴 수 없어요.
    꼭 미리보기 단계에서 명단을 확인하신 다음에 누르세요.


----------------------------------------
수동 메일 발송 (Ctrl + M, rtgreen 전용)
----------------------------------------

이 기능은 rtgreen 이라는 아이디로 로그인했을 때에만 써요.
다른 아이디로는 눌러도 안내 메시지만 나오고 닫혀요.

언제 쓰나요?

  자동 메일 발송이 어쩌다 실패했을 때 다시 보내거나,
  내가 직접 쓴 메일을 원하는 회원들에게 보낼 때 써요.

사용 방법:

  1. Ctrl + M 을 누르면 수동 메일 창이 열려요.

  2. "수신 대상" 라디오에서 고르세요.
       - 직접 입력
       - 최근 탈퇴된 회원
       - 최근 강등된 회원
       - 최근 승급된 회원
       - 준회원 전체 / 일반회원 전체 / 우수회원 전체 /
         최우수회원 전체 / 명예회원 전체 (v1.0 신규)
       - 레벨 5~9 전체 (v1.0 신규)
     (최근 작업·등급별 옵션은 해당 데이터가 있을 때만 나타나요.
      등급별 옵션이 안 보이면 Ctrl+F 로 회원 검색을 한 번 실행해
      회원 목록을 캐시한 뒤 다시 메일 창을 여세요.)

  3. 선택하면 수신 아이디, 제목, 내용이 자동으로 채워져요.
     그대로 써도 되고, 원하는 대로 고쳐도 돼요.

  4. (v0.5) 발송 방식은 "각 회원에게 개별 발송" 한 가지로 통일됐어요.
     수신자에게 다른 수신자 아이디가 절대 보이지 않아요.
     회원 수만큼 시간이 걸리지만(보통 회원 1명당 1초 정도) 안전해요.

  5. "발송" 버튼을 누르면 한 번 더 확인 창이 떠요.
     "예" 를 누르면 실제로 메일이 나가요.
     발송 도중 4분의 1, 절반, 4분의 3, 끝 시점에 음성으로 진행 상황을
     알려줘요.

  6. 발송이 끝나면 성공과 실패 건수가 화면에 나와요.

첨부파일 (v1.0 신규):
  발송 화면 아래에 "첨부파일" 영역이 있어요.
   - 추가(P)      : 파일 다이얼로그로 한 개 이상 파일 선택
   - 제거(V)      : 목록에서 선택한 파일 한 개만 제거
   - 전체 제거(Z) : 첨부파일 모두 비우기
  첨부한 파일은 모든 수신자에게 동일하게 함께 발송돼요.
  파일 이름과 용량이 목록에 표시되니 발송 전에 한 번 더 확인하세요.

등급 변경 이력 (Ctrl + Shift + Y) — v1.0 신규
----------------------------------------

지금까지 일어난 모든 등급 변경을 영구 보관해서 언제든 조회할 수 있어요.
"되돌리기" 와 별개로 동작하기 때문에 기록은 되돌리기 한 뒤에도 남아요.

저장 위치:
  data/level_history.db (로컬 SQLite)

기록되는 변경:
  · 자동 승급 (auto_promote)
  · 장기미접속 조정 (level_adjust)
  · 검색에서 수동 등급 변경 (manual)
  · 가입 승인 (approve)
  · 가입 거부 (reject)
  · 되돌리기 (undo)

조회 방법:
  1. 파일 메뉴 → "등급 변경 이력" 또는 Ctrl + Shift + Y
  2. 회원 아이디(부분 일치) 또는 유형 콤보로 필터링
  3. 위·아래 화살표로 한 건씩 들어보기

자동 알림 시점:

  · 예 → 한 명씩 보여주는 승인 화면이 열려요
  · 아니오 → 메인 화면으로 돌아가요. 나중에 작업 메뉴 →
            "신규 가입자 승인" 으로 언제든 다시 부를 수 있어요

승인 화면에서 할 수 있는 것:
  [A] 승인  → 등급을 준회원(레벨 5)으로 변경
              사이트에 즉시 반영. 실수해도 Ctrl+Z 로 되돌릴 수 있어요.
              · v1.0+: 승인 직후 회원에게 환영 메일이 자동으로 발송돼요
                (rtgreen 계정으로 로그인된 경우만. 다른 계정이면 건너뜀)
  [R] 거부  → 등급을 거부(레벨 2)로 변경
              사이트에 즉시 반영. 같은 회원은 다시 가입 신청해야 해요.
  [S] 미루기 → 이번에는 건너뛰고 다음 실행 시 다시 묻기
              "이미 본 적 있음" 으로 표시되지만 그래도 다시 알림돼요.
  [C] 끝내기 → 남은 회원은 그대로 두고 닫기

이미 한 번 본 회원도 다음 실행 시 또 알리고 싶다면
개발자 메뉴 → "신규 가입자 '본 적 있음' 기록 초기화" 를 누르면 돼요.

백업 비교 (Ctrl + Shift + D) — v0.4 신규
----------------------------------------

저장해 둔 분기 백업 두 개를 골라서 비교해요.
"이번 분기에 누가 새로 우수회원이 됐고, 누가 떨어졌나?"
같은 질문에 답하기 좋아요.

사용 방법:

  1. Ctrl + Shift + D 를 누르면 백업 비교 창이 열려요.
  2. "이전" 콤보와 "이후" 콤보에서 폴더를 골라요.
     기본값은 가장 최근 두 개로 미리 잡혀 있어요.
  3. "비교 실행(R)" 버튼을 누르면 결과가 나와요.

결과는 네 가지로 분류돼요:

  · 신규 진입 — 이전 백업에는 없었지만 이번엔 있는 회원
  · 승급      — 이전보다 등급이 올라간 회원
  · 강등      — 이전보다 등급이 내려간 회원
  · 빠짐      — 이전엔 있었지만 이번 백업에는 없는 회원

"TXT 내보내기(E)" 버튼을 누르면 backups 폴더 안에
diff_OLD_to_NEW_TIMESTAMP.txt 파일이 만들어져요.
스크린리더로 천천히 읽거나 운영보고에 붙일 수 있어요.

9. 안전 장치
==========================================

이 프로그램은 실수를 막기 위해 여러 안전 장치가 있어요.

  1. 동호회 관리자 아이디가 아니면 아예 열리지 않아요.

  2. 장기 미접속 조정은 항상 미리보기를 거쳐야 해요.

     한 번 더 묻는 단계가 있어요.

  4. 본인은 어떤 작업에서도 대상이 되지 않아요.

  5. 마지막 접속일을 확인할 수 없는 회원은 안전을 위해
     조정 대상에서 빠져요.

  6. 9레벨 명예회원과 동호회 관리자는 어떤 작업에서도
     등급이 내려가지 않아요.

  7. 모든 작업은 내부 로그 파일에 기록돼요.
     나중에 data 폴더 안의 logs 폴더에서 확인할 수 있어요.


==========================================
10. 문제가 생겼을 때
==========================================

"동호회 관리자 권한이 필요합니다" 라고 나올 때

  본인 아이디가 동호회 관리자 등급인지 확인하세요.
  또는 로그아웃 후 다른 관리자 아이디로 다시 로그인하세요.

"회원 목록을 분석하지 못했습니다" 라고 나올 때

  소리샘 사이트의 구조가 바뀐 것일 수 있어요.
  도움말 메뉴의 "관리자 페이지 덤프" 를 실행해 주세요.
  data 폴더 안의 dumps 폴더에 파일이 만들어져요.
  그 파일을 개발자에게 보내주세요.

네트워크 오류가 날 때

  잠시 후 다시 시도하세요.
  작업이 실패한 경우 "마지막 작업일" 이 갱신되지 않아서
  다음 실행 때 다시 시도할 수 있어요. 안전해요.

메일이 안 보내져요

  현재 로그인 아이디가 rtgreen 인지 확인하세요.
  rtgreen 이 아니면 메일 발송이 자동으로 꺼져 있어요.
  Ctrl + L 로 로그아웃한 뒤 rtgreen 아이디로 다시
  로그인하세요.


==========================================
11. 자주 하는 질문
==========================================

Q. 이 프로그램이 실수로 중요한 회원을 탈퇴시키지 않을까요?
A. 항상 미리보기 단계를 거치기 때문에 실수는 어렵습니다.
   또 본인, 명예회원, 관리자는 대상에서 빠져요.
   걱정되면 미리보기에서 "TXT 내보내기" 버튼으로
   명단을 저장한 뒤 천천히 확인하세요.

Q. 자동 로그인이 안 돼요.
A. Ctrl + L 로 로그아웃한 뒤 다시 로그인하세요.
   이번엔 "아이디와 비밀번호 저장" 체크박스를 꼭 체크하세요.

Q. 음성이 나오지 않아요.
A. NVDA 나 센스리더가 실행 중인지 확인하세요.
   프로그램은 두 스크린리더를 모두 지원합니다.

Q. 화면이 안 보여요 (창이 안 떠요).
A. Alt + Tab 키를 눌러서 창 목록을 확인해 보세요.
   백신 프로그램이 막았을 수도 있어요.

Q. 백업 파일을 보고 싶어요.
A. Ctrl + O 를 누르면 백업 폴더가 파일 탐색기에서 열려요.
